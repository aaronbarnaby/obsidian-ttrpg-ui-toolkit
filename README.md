# Obsidian TTRPG UI Toolkit

This plugin provides modern UI elements for playing popular TTRPG that provide building blocks for you to build
a beautiful markdown components for Obsidian.

> [!WARNING]
> This plugin is in early development, things are still open to change between versions.

## Game Systems

- Dungeons and Dragons
- Daggerheart

## Features

#### The following features are part of the original **DnD UI Toolkit**.

- General Components
  - Badges
  - Consumables
  - Spell Components
  - Stats

- Character Sheet
  - Ability Scores (Reskinned + Daggerheart Support)
  - Skills
  - Health Points
  - Event Buttons

- Dungeon Master
  - Initiative Tracker

#### The following are changes / additional features

- Dice roller (including Duality Dice for Daggerheart)

## Planned Work

- Continue Reskinning and adding Daggerheart support for original components
- Improved Initiative Tracker
- Monster / Adversary Builder
